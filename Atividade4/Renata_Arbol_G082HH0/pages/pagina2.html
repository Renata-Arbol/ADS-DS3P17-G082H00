INTRODUÇÃO

Um documento HTML tem uma estrutura que lembra um contrato ou uma petição. Há um início (<html>), um cabeçalho (<head>) e um corpo (<body>). No cabeçalho, você coloca informações sobre a página, como um título, que é como um preâmbulo de um contrato. No corpo, você coloca o conteúdo visível da página, que é como as cláusulas e termos do contrato.

Dentro do corpo, são usadas diversas “tags” para formatar o texto, criar listas, adicionar links e imagens, assim como fazemos com diferentes partes de um documento jurídico. Também existem atributos, que são como notas de rodapé que fornecem informações adicionais sobre um elemento, como especificar o endereço de um link.

AS TAGS DO HTML 5

O HTML é usado com o CSS, que controla a aparência da página, como se fosse a formatação de um documento jurídico, e o JavaScript, que adiciona funcionalidade e interatividade, como um anexo com informações adicionais.

Dessa forma, seguem alguns exemplos de “tags” de HTML e a respectiva funcionalidade que representa na elaboração de uma página web:

  • <html>: Define que o documento é um arquivo HTML.
  • <!DOCTYPE>: Define o tipo de documento.
  • <a>: Cria um link. Atributos: href, target, download.
  • <abbr>: Abreviação.
  • <address>: Informação de contato.
  • <area>: Área dentro de um mapa de imagem. Atributos: shape, coords, href.
  • <article>: Conteúdo independente.
  • <aside>: Conteúdo à parte.
  • <audio>: Som ou música. Atributos: src, controls, autoplay.
  • <b>: Texto em negrito.
  • <base>: URL base para links. Atributo: href.
  • <bdi>: Isola texto com direção diferente.
  • <bdo>: Sobrepõe direção de texto. Atributo: dir.
  • <blockquote>: Citação longa. Atributo: cite.
  • <button>: Botão. Atributos: type, disabled.
  • <canvas>: Desenho gráfico. Atributos: width, height.
  • <caption>: Legenda de tabela.
  • <cite>: Título de obra.
  • <code>: Código de computador.
  • <col>: Grupo de colunas em tabela. Atributo: span.
  • <colgroup>: Grupo de colunas em tabela. Atributo: span.
  • <head>: Contém informações sobre a página.
  • <title>: Define o título da página.
  • <body>: Contém todo o conteúdo visível da página.
  • <h1> até <h6>: Cabeçalhos. O número indica o nível do cabeçalho.
  • <p>: Define um parágrafo.
  • <data>: Valor legível por máquina. Atributo: value.
  • <datalist>: Lista de opções para input.
  • <dd>: Descrição de termo em lista de definição.
  • <del>: Texto deletado. Atributos: cite, datetime.
  • <details>: Detalhes que o usuário pode abrir. Atributo: open.
  • <dfn>: Termo de definição.
  • <dialog>: Caixa de diálogo.
  • <dl>: Lista de definição.
  • <dt>: Termo em lista de definição.
  • <em>: Ênfase de texto.
  • <embed>: Conteúdo incorporado. Atributos: src, type, width, height.
  • <fieldset>: Grupo de campos de formulário.
  • <figcaption>: Legenda de figura.
  • <figure>: Conteúdo relacionado à imagem.
  • <footer>: Rodapé de seção.
  • <form>: Formulário. Atributos: action, method.
  • <ul>: Lista não ordenada.
  • <ol>: Lista ordenada.
  • <li>: Item em uma lista.
  • <img>: Insere uma imagem. Atributo: src.
  • <br>: Quebra de linha.
  • <header>: Cabeçalho de seção.
  • <hgroup>: Grupo de cabeçalhos.
  • <hr>: Linha horizontal.
  • <div>: Define uma seção.
  • <i>: Texto em itálico.
  • <iframe>: Frame incorporado. Atributos: src, width, height.
  • <input>: Campo de entrada. Atributos: type, name, value, e outros.
  • <main>: Conteúdo principal.
  • <map>: Mapa de imagem. Atributo: name.
  • <mark>: Texto destacado.
  • <menu>: Lista de comandos.
  • <meta>: Metadados sobre documento. Atributos: name, content.
  • <meter>: Medida escalar. Atributos: value, min, max.
  • <nav>: Navegação.
  • <noscript>: Conteúdo alternativo se JavaScript estiver desativado.
  • <object>: Objeto incorporado. Atributos: data, type, width, height.
  • <span>: Define uma parte do texto.
  • <table>: Cria uma tabela.
  • <thead>, <tbody>, <tfoot>: Definem a cabeça, o corpo e o rodapé de uma tabela.
  • <tr>: Define uma linha em uma tabela.
  • <th>: Define uma célula de cabeçalho em uma tabela.
  • <td>: Define uma célula de dados em uma tabela.
  • <param>: Parâmetro para objeto. Atributos: name, value.
